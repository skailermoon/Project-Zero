Project Zero is a hardcore zombie apocalypse modpack built around scavenging, survival, exploration, and fighting against overwhelming undead hordes. Civilization has collapsed, resources are scarce, and every day is a battle to stay alive. Multiplayer is supported. All players should use the same version of the pack and provided configurations to ensure compatibility.

## Survival Tips
Avoid unnecessary fights early game.<br>
Establish a secure shelter before nightfall.<br>
Never underestimate a horde.<br>

## Recommended Specifications
8 GB RAM Minimum<br>
12-16 GB RAM Recommended<br>
Prism Launcher Recommended<br>

## Installation
Download or clone the latest release.<br>
Copy the files into a new Prism Launcher instance.<br>
Launch the game.<br>
Survive.<br>
