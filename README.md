# Project Zero
ProjectZero is a hardcore zombie apocalypse modpack built around scavenging, survival, exploration, and fighting against overwhelming undead hordes. Civilization has collapsed, resources are scarce, and every day is a battle to stay alive.

## Multiplayer
Multiplayer is fully supported. All players should use the same version of the pack and configurations to ensure compatibility.

## Survival Tips
Avoid unnecessary fights early game.
Establish a secure shelter before nightfall.
Never underestimate a horde.

## Recommended Specifications
8 GB RAM Minimum
12-16 GB RAM Recommended
Prism Launcher Recommended

## Installation
Download or clone the latest release.
Import or copy the files into a Prism Launcher instance.
Launch the game.
Survive.
